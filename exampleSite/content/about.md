---

This site demonstrates **Felmdrav**, a Hugo theme based on **Bootstrap 5** with optional **Bootswatch** support.

The example site is intended as a **theme showcase**. It demonstrates common layouts, components, and configuration options to help you understand how the theme behaves and how it can be customized.

### Documentation and source code

The complete documentation, configuration reference, and source code are available on GitHub:

- https://github.com/geschke/hugo-felmdrav

### Notes

This example site is not meant to represent a complete website.  
Its purpose is to provide visual and structural reference for theme users.
